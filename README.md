# Java-Programming-Journey
##Java Calculator project

A simple console-based calculator built in Java as part of my Java learning journey at BRAC University.

##Features


Performs addition, subtraction, multiplication, and division
Extracts and greets the user by their email username
Menu-driven interface — choose the operation you want
Input validation — invalid choices prompt the user to try again
Handles multiple calculations in a single run (user-defined count)


##How to Run


Make sure Java is installed on your system
Compile the program:
Follow the on-screen prompts


### What I Learned

This project helped me practice:


Java's Scanner class for user input
String methods (substring, indexOf, toUpperCase)
Switch expressions (Java's modern arrow syntax)
Writing modular code using separate methods for each operation
Debugging — specifically tracking down mismatched braces
